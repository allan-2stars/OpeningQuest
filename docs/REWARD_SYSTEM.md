# Opening Quest - Reward System

## Philosophy

Rewards support learning.

Rewards must never replace learning.

## Reward Categories

1. XP
2. Keys
3. Cosmetics
4. Achievements
5. World Rewards

## Cosmetics

Categories:
- Piece skins
- Board themes
- Mascot costumes
- Profile frames
- Titles

## Piece Skins

Customizable:
- Pawn
- Knight
- Bishop
- Rook
- Queen
- King

Themes:
- Forest
- Royal
- Defender
- Dragon
- Ice
- Wizard
- Pirate
- Space
- Grandmaster

## Achievements

Categories:
- Learning
- Mastery
- Consistency
- Exploration
- Collection

Examples:
- First Lesson
- First Perfect Run
- First Mastered Lesson
- 7-Day Streak
- Italian Master
- Caro-Kann Defender
- First Full Chess Set

## Monetization Rule

V1 has no monetization:
- No ads
- No subscriptions
- No in-app purchases
- No loot boxes
