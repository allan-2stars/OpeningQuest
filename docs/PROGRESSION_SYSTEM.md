# Opening Quest - Progression System

## Systems

Opening Quest uses:
- XP
- Keys
- Mastery
- Reviews
- Achievements

## XP

XP shows lifetime experience.

XP never decreases.

XP does not unlock lessons.

Rewards:
- Correct move: +1 XP
- Perfect run: +25 XP
- Lesson mastered: +100 XP
- Review success: +50 XP
- Boss victory: +250 XP
- World completion: +500 XP

## Keys

Keys unlock content.

Rewards:
- Lesson mastered: +1 Key
- Boss victory: +3 Keys
- World completion: +5 Keys

Keys can unlock:
- Worlds
- Special lessons
- Cosmetics
- Future expansion content

## Mastery

Mastery is tracked per lesson.

Mastery can increase and decrease.

Requirement:
- 10 perfect runs = mastered

Decay:
- 2 failed reviews = mastery drops one level

## World Progression

World completion requires:
- Required lessons completed
- Boss battle passed
- Required keys available if applicable

## Streak

A streak increases when the player completes at least one lesson in a day.

Missed days may be protected by Streak Freeze in future.
