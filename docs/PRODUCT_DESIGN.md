# Opening Quest - Product Design

## Vision

Opening Quest is a chess adventure game that teaches opening repertoires through repetition, mastery, visual progression, and long-term review.

The product combines:
- Chess opening training
- Adventure map progression
- Rewards and collectibles
- Habit-building systems
- Kid-friendly presentation

The goal is not to teach chess theory in a textbook way.

The goal is to make opening moves become instinctive.

## Product Positioning

Opening Quest is not:
- A chess engine
- A chess analysis tool
- A chess database
- A social chess app
- An online multiplayer platform

Opening Quest is:
- A chess opening mastery game
- A guided repertoire builder
- A memory and habit training system

## Target Audience

Primary:
- Kids aged 8-14 learning chess

Secondary:
- Adult beginner/intermediate chess players

## Experience Modes

### Adventure Mode

Default mode.

Includes:
- World maps
- Mascot
- Reward unlocks
- Progression path
- Boss battles

### Classic Mode

Minimal training interface.

Includes:
- Practice
- Mastery
- Statistics
- Review queue

Adventure Mode and Classic Mode use the same training engine.

## Core Loop

Learn
Practice
Repeat
Perfect Run
Master
Unlock
Review
Retain

## V1 Scope

Must have:
- Adventure Mode
- Classic Mode
- Opening lessons
- White openings and Black defenses
- Practice board
- Guided Mode
- Instinct Mode
- Mastery tracking
- Unlock progression
- XP
- Keys
- Badges
- Piece skins
- PGN import/export
- Custom openings
- Offline storage

Must not have:
- Multiplayer
- Cloud sync
- AI coach
- Stockfish
- Social features
- Parent dashboard
- Monetization
